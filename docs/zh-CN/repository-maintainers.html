仓库维护人员注意事项

贡献者应该在被发现贡献后立即被识可，以确保他们的努力不会在往后更新列表时被忽略。 对于不提交到仓库的贡献会很难，但是尽最大努力减少贡献和更新列表之间的时间。

专业提示：如果你的项目是@all-contributors bot 🤖，简单地在issue里写个评论或者pull request来识别他们的贡献。 例如：

@all-contributors please add @tbenning for design ✨

作为仓库的所有者或者维护人员并不意味着你独自负责保持贡献者列表的最新状态。 你应该鼓励贡献者尽可能地将他们自己添加到列表中。 这可以是对问题，博客或答案的评论，或者在适当情况下通过更直接的形式沟通。

许多贡献者可能没有意识到他们的努力足以在贡献者名单中得到认可。 这可能是因为他们没有阅读或者理解本规范中规定的贡献者定义，或者是因为他们觉得它不够重要。 在这些情况下，你仍然应该鼓励他们添加自己，但是你可能需要自行添加贡献者 (虽然以pull request的形式做是一个好主意，以确保他们同意被添加)。

最后，对于一个贡献添加到列表中时，没有硬性和快速的规则。尽力做到公平，并且确保所有贡献者都得到认可。
